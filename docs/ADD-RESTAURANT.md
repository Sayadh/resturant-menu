# 🏪 Նոր հաստատություն ավելացնելու ուղեցույց (runbook)

Այս փաստաթուղթը քայլ առ քայլ ցույց է տալիս՝ ինչպես ավելացնել նոր հաստատություն (ռեստորան/սրճարան/բար) **menus.am**-ում՝ super-admin-ից մինչև պատրաստ, տպելու QR կոդ։

> Հրապարակային մենյուն միշտ հասանելի է՝ `https://menus.am/<slug>`

---

## 0. Նախապայմաններ

- **Super-admin հաշիվ** (platform-level)։
- DB-ում **plan-ները** ստեղծված են՝ `free` (Starter), `pro` (Professional), `business` (Business)։
- Supabase Storage-ի `menu-images` bucket-ը **Public** է (նկարների համար)։

---

## 1. Ստեղծել հաստատությունը (super-admin)

1. Մուտք գործիր որպես **super-admin** → `/admin` → բացվում է **Restaurants** (super-admin) բաժինը։
2. Սեղմիր **«Նոր հաստատություն»**, լրացրու․
   - **Slug** — URL-ի մասը (օր․ `tun-lahmajo`) → կայքը կլինի `menus.am/tun-lahmajo`։ Միայն փոքրատառ, latin, առանց բացատի (`-`-ով)։
   - **Անուն** — հաստատության անունը։
   - **Թեմա** — Aria / Atelier / Maison / Heritage (հետո owner-ը կարող է փոխել)։
   - **Լռելյայն լեզու** — Հայերեն / Русский / English։
   - **Owner email** + **password** — հաստատության տիրոջ մուտքի տվյալները։
3. Սեղմիր **Ստեղծել**։ Համակարգը ինքնաշխատ ստեղծում է․ հաստատությունը + 3 լեզու + 3 default բաժին (**Ուտեստներ / Ըմպելիքներ / Ալկոհոլ**) + owner user։
4. **📌 Պահիր owner-ի email/password-ը** — հաստատությանը կփոխանցես։

---

## 2. Նշանակել փաթեթը (plan)

Super-admin ցուցակում → հաստատության **Խմբագրել** → **Փաթեթ** → ընտրիր **Starter / Professional / Business** → Պահպանել։

- **Starter (free)** — 2 լեզու, մինչև 250 ապրանք, 1 թեմա, առանց AI։
- **Professional (pro)** — անսահմանափակ ապրանք, 3 լեզու, բոլոր թեմաները, **AI թարգմանություն + AI նկարագրություն**։
- **Business (business)** — Professional + սեփական դոմեն, բազմաթիվ օգտատերեր, վիճակագրություն։

> ⚠️ AI կոճակները admin-ում երևում են **միայն Professional/Business**-ի դեպքում։

---

## 3. Owner-ով մուտք → «Հաստատության մասին»

Դուրս արի super-admin-ից, մուտք գործիր **owner-ի** տվյալներով → `/admin` → **Հաստատության մասին**․

- **Անուն**
- **Tagline / կարգախոս** — 3 լեզվով (հայ/рус/eng)
- **Logo** — վերբեռնիր (երևում է մենյուի header-ում)
- **Հասցե**, **աշխատանքային ժամեր**, **rating** (ըստ ցանկության)

Պահպանիր։

---

## 4. Ընտրել թեման

**Դիզայն** բաժնից ընտրիր թեման (Aria / Atelier / Maison / Heritage)։ Փոխվում է մեկ սեղմումով, մենյուն չի կորչում։

---

## 5. Բաժիններ (Sections)

3 default բաժինն արդեն կա (Ուտеստներ / Ըմպելիքներ / Ալկոհոլ)։ Կարող ես խմբագրել/ջնջել/ավելացնել։ Ամեն բաժնի համար․

- **Անուն** — 3 լեզվով
- **Իկоն** — վերբեռնիր փոքր նկար (**200×200px**, քառակուսի) — երևում է բաժինների tab-ներում (Aria/Heritage)։ Եթե նկар չկա՝ էմոջին է։
- ▲▼ կոճակներով դասավորիր հերթականությունը։

---

## 6. Կատեգորիաներ

**Մենյուի կառուցում** → ընտրիր բաժինը → **Ավելացնել կատեգորիա**․

- **Անուն** + **նկարագրություն** — 3 լեզվով
- **Իկоն** (փոքր, tab-ի/banner-ի կլոր icon) — **400×400px**
- **Դեսքթоփ նկар** (մեծ banner) — **1600×500px**
- **Մոբайլ նկар** (banner մոբայլում) — **800×600px**
- ▲▼-ով դասավորիր հերթականությունը

> Եթե նկар չդնես, banner-ը կվերցնի առաջին ապրանքի նկարը (fallback)։

---

## 7. Ապրանքներ

**Ապրանքներ** → **Ավելացնել ապրանք**․

- **Կատեգорիա** (պարտադիր)
- **Անվանում** + **նկարագրություն** — 3 լեզվով
  - **✨ AI նկարագրություն** — անունից ավтоմат գեներացնում է նկարագրություն (Pro/Business)
  - **🌍 Թարգմանել** — մեկ լեզվից լրացնում է մյուսները (Pro/Business)
- **Գին** (֏)
- **Նկар** — վերբեռնիր (ավтоմат compress + WebP)
- **Badges** (Հիթ / Նոր / Խորհурդ / Կծու / Վեган / Մատչելի)
- **Հասանելի** — toggle (առկա / սպառված)
- ▲▼-ով դասավորիր հերթականությունը

---

## 8. Հերթականություն

Ամեն ցուցակում (բաժիններ/կատեգորիաներ/ապրանքներ) ▲/▼ կոճակներով դասավորում ես, թե **որը որ հերթին երևա**։ Նույն հերթականությունը կերևա հրապարակային մենյուում։

---

## 9. QR կոд

**QR կоդ** բաժնից՝ պատճենիր հղումը կամ ներբեռնիր QR-ը → տպիր ու դիր սեղանների վրա։

---

## 10. Ստուգում ✅

- Բացիր `https://menus.am/<slug>` → մենյուն երևո՞ւմ է։
- Փոխիր լեզուն (հայ/рус/eng) → թարգմանությունները ճի՞շտ են։
- Մոբայլում ստուգիր banner-ներն ու icon-ները։
- (SEO) View Source → պիտի լինի `<title>`, description, ու `Restaurant` JSON-LD։

## 🧰 Խնդիրների լուծում (troubleshooting)

| Խնդիր | Պատճառ / լուծում |
|---|---|
| AI կոճակները չեն երևում | Plan-ը `free`/Starter է → super-admin-ից դարձրու Professional/Business |
| AI-ը «Monthly limit reached» | Ամսական 1200 call-ի սահմանն է լրացել |
| Նկարը 404 / չի բացվում | `menu-images` bucket-ը Public չէ → Supabase → bucket → Settings → Public |
| Նկար չի բարձրանում | Backend-ի `.env`-ում `SUPABASE_URL / SUPABASE_SECRET_KEY / SUPABASE_BUCKET` դատարկ են |
| Կատեգորիա ավելացնել չեմ կարող | Նախ պիտի բաժին լինի ընտրված |
| «Cross-restaurant access» error | restaurantId-ն JWT-ից է — owner-ը միայն իր հաստատությունն է կառավարում |
